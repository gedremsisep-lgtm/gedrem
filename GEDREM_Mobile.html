ℹ Como instalar / usar
Instalar como aplicativo: abra este arquivo no Chrome do celular → menu ⋮ → "Adicionar à tela inicial". O ícone GEDREM aparece como um app normal.

Usuários e acesso: Admin, Técnico A e Técnico B veem todas as demandas de todas as equipes; cada encarregado vê apenas as da própria equipe. PIN inicial de todos: 1234 — cada um deve alterar o seu em Config.

Distribuir para as equipes: envie este mesmo arquivo por WhatsApp para cada encarregado; cada um entra com o próprio usuário e recebe a própria programação.

Fluxo diário: 1) importar programação (ou lançar manual) → 2) no local, capturar GPS (endereço sai por extenso e o quadro de mapa entra nas fotos) → 3) fotografar → 4) concluir → 5) gerar PDF e enviar ao fiscal.

GPS/endereço: precisa de internet no momento da captura para converter as coordenadas em endereço e montar o mapa. Sem internet, as coordenadas ficam salvas e você pode capturar o endereço depois.

Conexão com o sistema: automática — o app lê a Programação do sistema (cada equipe só vê a sua) e, ao concluir uma demanda, o relatório com fotos entra sozinho no Relatório Fotográfico do sistema com o próximo código sequencial. A demanda correspondente na Programação vira "Executado".
